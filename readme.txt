== Description ==

For more information about Qody or it's various prestigious workers, head on over to http://plugins.qody.co for the latest scoop.

== Installation ==

1. Upload `qodys-owlnest` to the `/wp-content/plugins/` directory
2. That's it!  There's no need to activate this plugin, as the framework simply has to "exist" on the server to function
